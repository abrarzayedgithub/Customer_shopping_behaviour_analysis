# Customer_shopping_behaviour_analysis
# 🛍️ Customer Shopping Behaviour Analysis

A complete end-to-end data analysis project exploring customer purchasing patterns, subscription behaviour, and revenue trends across different demographics and product categories.

---

## 📌 Project Overview
This project analyzes a retail customer dataset to uncover actionable business insights around:
- Which customer segments generate the most revenue?
- How does subscription status affect purchase behaviour?
- Which product categories and age groups drive the most sales?
- What shipping preferences do customers have?

---

## 🗂️ Project Structure

| File | Description |
|------|-------------|
| `customeer_shopping_behavior_analysis.ipynb` | Python EDA — data cleaning, analysis, visualizations |
| `SQLQuery1.sql` | SQL queries for data exploration and aggregation |
| `customer_behavior_dashboard.pbix` | Interactive Power BI dashboard |
| `Business Problem Document.pdf` | Problem statement and project objectives |
| `Customer Shopping Behavior Analysis.pdf` | Full analysis report with findings |

## 🔍 Key Insights

1. **Subscription gap** — Only 40.81% of customers are subscribed. There is significant upsell opportunity among the 59.19% non-subscribers.
2. **Age-based revenue** — Middle-aged customers contribute the most revenue, while Young Adults have the lowest — suggesting targeted promotions could boost younger segment spending.
3. **Category focus** — Outerwear dominates both revenue and sales volume for subscribed male customers.
4. **Shipping preference** — Standard and Free Shipping are the most preferred, which can inform logistics and discount strategy.
